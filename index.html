Бесплатный сервис для хранения данных. Все устройства (ваш телефон, телефон жены, компьютер) будут видеть одни данные в реальном времени.
1
Зайдите на jsonbin.io и зарегистрируйтесь (бесплатно)
2
В личном кабинете перейдите в API Keys и скопируйте ключ
3
Вставьте ключ в поле ниже и нажмите «Создать хранилище»
4
Скопируйте полученный Bin ID и введите его на других устройствах
