☁️ Синхронизация
Firebase — бесплатная облачная база
Все устройства — ваш телефон, телефон жены и компьютер — видят одни данные в реальном времени. Настройка занимает 3 минуты.
1
Зайдите на console.firebase.google.com → войдите через Google → нажмите Создать проект
2
Введите любое название проекта (например finance-planner), аналитику можно отключить → создайте проект
3
В меню слева: Сборка → Realtime Database → Создать базу данных → выберите Начать в тестовом режиме
4
Скопируйте URL базы данных — он выглядит как https://finance-planner-abc-default-rtdb.firebaseio.com
5
Вставьте URL в поле ниже и нажмите «Подключить». На других устройствах введите тот же URL.
💡 Тестовый режим Firebase работает бесплатно 30 дней. После этого нужно настроить правила доступа — напишите мне, помогу.
